# ToDo App - Personal Goals & Tasks

## Description
This is a web application developed with React that allows users to manage personal goals and tasks (To-Do List).
Users can add tasks with a name, description, and due date, and remove them when completed.

---

## 🚀 Features
- Add new tasks/goals
- Delete tasks
- Responsive design (desktop and mobile)
- Clean interface using React Bootstrap

---

## Technologies Used
- React
- React Bootstrap
- CSS
